🚀 To host this on GitHub Pages:
  1. Create a free account at github.com
  2. Click New repository → name it nq30-watchlist → set to Public
  3. Upload this HTML file, renaming it to index.html
  4. Go to Settings → Pages → Source: Deploy from branch → branch: main → Save
  5. Your live URL will be: https://yourusername.github.io/nq30-watchlist
